# xmrig-installer


## Installation Instructions

To set up the XMRig Installer, execute the following command from your server's SSH terminal:

```bash
bash <(curl -s https://raw.githubusercontent.com/anuragsavaliya1/xmrig-installer/main/install-mo.sh)
```

This command will automatically download and execute the installation script, ensuring everything is set up seamlessly and efficiently. No additional configuration is required.
